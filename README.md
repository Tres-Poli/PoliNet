<h1>PoliNet</h1>

This is a pet project covering such aspects of game development as:
- Cross layer messaging;
- Client/Server communications;
- Latency compensation methods;
- Runtime patterns (FSM);
- UI;
- Clean architecture;
- Code generation for ECS aspects modules and network messages;
- ECS utilization;
- Runtime assets management;
- IoC container utilization;
